西元二〇〇八年，三月五日，我做与摩托车上，向广场码头前去。我无法不注意到，在老市区昏黄的老厝边，那轮通红的夕阳，空悬于天边，震慑着我的心灵。 多久没有这么凝视那一轮通红的夕阳了。虽说金中位于礐石山西坡，是很适宜看夕阳的。但或许是因为平日的繁忙，抑或是汕头港南岸的水土不足以勾起我心中的感动吧。在金中很少有欣赏夕阳。而每个星期仅有的在汕头的一天，也忙于享受从学习机器还原为自由人，有披着面具的人还原为有着真实情感的真我的感觉，也很少有机会去凝视那一轮红日。 // 上次这么凝视夕阳是在什么时候？ 1997，我与母亲，坐于澄海人民公园的石凳上。东方，皎洁的月娘缓缓上升，伴随着西方那轮红日缓缓落下。 2000，与阿姐一起坐在母亲摩托车的后座上，一起吧路过的学校墙壁上的校训串起来唱城歌——虽然一回到家又会像往常一样打起架来。 2006，踏着自己的脚车，自由地穿梭于市区的大街小巷，望着前方，那轮红日，怀揣着心中的爱与梦想。 // 又很震撼于《星球大战》中的Tatooine星上的两个太阳同时落下的场景（虽然在我正在震撼中的时候我妈很没情趣地说，上面那个太阳明显是假的）——看这么壮丽的景观，总是会感到自己的渺小，自己不过生活在，银河系中千千万万个恒星中的一个横星里面一个普通得不能再普通的行星上罢了。总感叹生命的短暂。再漫漫的历史长河中，我们又能经历其中的几分呢。 // 不知道几岁的时候知道有死亡。当时的我，还小，不知道才来到这世上经历了多少个春夏秋冬，当时总对死亡怀有极大的恐惧，总害怕有一天死亡的到来。总害怕，难得降临到这个世上，又终要离开…… // 似乎从哪里听说，有时间旅行的存在。有值得我们难免会想进行一番时间旅行。想回到历史上的某个时刻，以一己之力改变历史。 后来的我，不那么天真幼稚，后来，又有了很多烦恼，便想过，回到过去。曾经一段时间沉浸在周杰伦的《回到过去》之中：想回到过去，试着让故事继续，至少不再让你离我而去…… // 但我们真的想回到过去吗？曾经思考过，RPG游戏中的S/L（保存与读取）大法与人生。玩RPG时一直储存游戏，其实有多少次真正读取过多少次进度呢，还不都是为了防止突然停电之类的事故导致的进度消失。怕辛辛苦苦打下来的进度失去了。 游戏尚且如此，那人生呢？谁又愿意放弃此时所拥有的而回到过去呢？ 我愿意回到什么时候？回到幼儿园与母亲一起在人民公园跑步的时刻；回到小学时的仲夏练完书法与妈妈坐在运行中的空调下面欣赏阿姐弹奏的钢琴曲的时刻；回到小学时的仲夏在开着冷气的房间里午睡醒来时母亲端来一杯鸡蛋汤的时刻；回到小学时一个暑假的清晨外婆带我到还带着农村气息的别墅群中的小道漫步的时刻；回到小学时与苏嘉盛，董煜基一起谈天说地，一起快乐地踢毽子，扔纸团的时刻；回到初中时在冬天顶着寒风骑车向学校前去的时刻（见旧文[原创精品]怀念那美丽的飞厦的一切）；回到高一上学期时她向我微笑的时刻…… // 可是，如果回到了当时那些幸福时刻，是不是就会失去之后那些幸福呢？如果回到当时，我是不是会失去现在爱我的人和我爱的人呢？我是不是会失去我现在的性格和思想呢？ 也许你会说，失去了这些幸福，我也许会得到更多幸福。我也许会有更成熟的性格与思想。但如果这样，那那个人还是不是我？不知道。一个老人回到出生的时候，经历一次重生，那么它与死亡有什么区别。一个人的记忆，一个人的幸福，一个人的思想，一个人的爱与善良失去了，那么这个人与死又有什么差别？ // 曾经与洁纯谈论关于莫泊桑的《项链》中的马蒂尔白费十年青春的看法。我们的峥嵘文学社副社长坚持为马蒂尔是趋势年的光阴而惋惜，但我却认为，无论是什么经历，对于一个人，都不能算是白费。 // 我总是强迫许多不堪回首的镜头不在自己的脑海中重演，但他们也算是我生活中的一种历练吧，正是他们构成了今天的我吧。就像普希金的《假如生活欺骗了你》里面所说的：而那过去了的，就会成为亲切的回忆…… 那过去的一幕幕在脑海中重演，无法抗拒。不知为何，过去的快乐与幸福，悲伤与尴尬，愤怒，到现在都化作嘴边的一抹微笑。无论如何，你必须去接受自己的过去——接受过去，也便是接受自己。何况，过去还是你的一份不可多得的财产…… 无论过去如何，我都不会去逃避过去，至少不欺骗自己。我会摆脱过去的光环与阴影，努力而乐观地生活下去。 // 后记：最近在自学曼昆的《经济学原理》 。当时我对洁姐说，学经济学有实际意义一点，不想哲学都是在无病呻吟。可是不知不觉我却在自己的文章中讨论哲学问题了…… Update: TMD yupoo把我的相片都删除了，Tatooine的落日看不到了，但是附上一幅：也就是文章开始那个场景 汕头的落日
