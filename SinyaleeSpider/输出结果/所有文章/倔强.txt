最近，有一件事情。 我不应该传播这件事情的…… Chapter 1: What happened? 午休_羽毛球 黑影_巨响_尖叫 人来_人往 血 泪 Chapter 2: Why? 我听到的第一个版本，也是官方版本是说，某人为了假飒，坐到栏杆上，结果一不小心，就翻过去了…… Chapter 3: Other facts. 1 ) 学校的走廊的栏杆都是有护栏的，坐在上面又不舒服又危险。 2 ) 当时是午休时间，并没有多少人，根本就没有对象可以来假飒。 3 ) 作为一个初二学生，除非上公开课，很少去那边那一栋楼，更不要说从二三楼，爬到四五楼去玩？ 4 ) 期中考刚过。 5 ) 之后学校不再发考试成绩单。 6 ) 她的父母（曾经）是学校的老师。 7 ) 学校不知为何心虚，封锁消息。 8 ) 家长暂时没有要求赔偿（对汕头人来说是不寻常的）。 Chapter 4: Why? 以下只是听说，不确定： 1) 其班主任是历史朱……据说为人巨苛刻，经常把小事情上纲上线，十分伤人自尊心。 2) 她是找关系进飞厦的，期中考试垫底。 但是，作为一个飞厦的优秀毕业生，我应该有能力做出自己的判断的。 Chapter 5: My feelings. 在河畔乱逛，看到2篇文章：sth. about 飞厦，孩子。请不要伤害自己，很伤感。 这种事情天天有发生，却只有这件事让我伤心不已。 或许确实是这样子——我们同根生。我们有流着一样的血，怀着一样的梦；我们在一个校园里面，一样的场地，一样的天空；一样的流泪，一样的欢笑。 翻出三月份回飞厦照的相片，真的很很感慨。 多么好的青春，多么好的岁月。我宁愿相信，你看到这风景，是想要来一个深情的拥抱： 谁会想到是那样子呢，谁又想要这样子呢？谁又忍心看到有人这样子呢？ 一个初二的学生，被逼到这个地步，要承受多大的压力？哪个老师又忍心说，人是逼出来的？ Chapter 6: 倔强 我和我最后的倔强 握紧双手绝对不放 下一站是不是天堂 就算失望不能绝望 我和我骄傲的倔强 我在风中大声的唱 这一次为自己疯狂 就这一次我和我的倔强 自从GDOI结束后发现这首歌，发现，无论什么时候听到这首歌，都是莫大的鼓励。 无论现在情况如何，反正，将来的路都很长。答应我，坚强地走下去，好吗？ 让我们，握紧双手，绝对不放。 但愿木棉花永远火红…… Chapter 7: Rethink 好近 飞厦，好近…… 到底我离她多远？ 1 ) 走后门进飞厦的？有了 2 ) 家长是老师？有了 3 ) 喜欢把小事上纲上线的班主任？没有是不可能的 到底还差多远？ 1 ) 我成绩还过得去。虽然大人是不会知足的…… 2 ) 我脸皮稍微厚一点。 封锁消息，只能是欲盖弥彰——至少金中的同学都知道了，我妈妈也从她朋友处知道了。 谁跟我提“学校形象”，我跟谁过不去！ 什么学校形象，哪里有同学的健康快乐重要！我对飞厦的感情，就像我对这片土地感情一样：我热爱这里的人，在这里度过的青春岁月。我对所谓“学校”抑或是“国家”这个人造的概念没有任何感情。 如果不面对，而是说什么“意外”，估计某一天，我在金中的日记会变成： 热_小卖部_冰淇淋 黑影_巨响 尖叫_车来_车往 血 泪 隔壁不是李嘉诚医院……
